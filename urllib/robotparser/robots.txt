# Robots协议也称作爬虫协议、机器人协议，它的全名叫做网络爬虫排除协议（Robots Exculusion Protocol),用来告诉爬虫和搜索引擎哪些页面可以抓取，哪
些不可以抓取。通常是一个叫做robots.txt的文本文件，一般放在网站的根目录下。
User-agent: *  # 搜索爬虫的名称， * 表示对所有爬虫都有效
Disallow: /  # 制定了不允许爬取的目录
Allow: /public/  # Allow 一般和 Disallow 一起使用，一般不会单独使用，用来排除某些限制。现在我们设置为 /public/，则表示所有页面不允许抓取，
但可以抓取 /public/目录

# 上面的例子实现了对所有搜索爬虫只允许爬取pulic目录的功能
# 常见的例子

# 1 禁止所有爬虫访问任何目录的代码如下：
User-agent: *
Disallow: /

# 2 允许所有爬虫访问任何目录的代码如下:(robots.txt留空也可以实现这样的功能）
User-agent: *
Disallow:

# 3 禁止所有爬虫访问网站某些目录的代码如下：
User-agent: *
Disallow: /private/
Disallow: /tmp/

# 4 只允许某一个爬虫访问的代码如下：

User-agent: WebCrawler
Disallow:
User-agent: *
Disallow: /
